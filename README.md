# buffetIKIT

Буффет ИКИТ.

Команда и роли:
Дронгаль Наталия – Руководитель проекта, Дизайнер, Back-end
разработка.
Конохова Юлия – Front-end разработка, Бизнес-аналитик.
Необходимо указать, что роли back и front могут меняться с течением
времени, поскольку навыки разработки у участников команды разные, и не
исключена возможность помогать друг-другу в той или иной задаче.
Ресурсы, технологии:
1) Flutter – Комплект средств разработки и фреймворк с открытым исходным
кодом для создания мобильных приложений под Android и iOS, вебприложений, а также настольных приложений под Windows, macOS и
Linux.
2) Язык программирования Dart – Язык программирования, созданный
Google. Dart позиционируется в качестве замены/альтернативы JavaScript.
3) Visual Studio Code – Текстовый редактор, разработанный Microsoft для
Windows, Linux и macOS. Позиционируется как «лёгкий» редактор кода
для кроссплатформенной разработки веб- и облачных приложений.
4) Эмулятор – Эмулятор Android позволяет создавать виртуальные
устройства Android на ПК. Иначе говоря, вы можете запускать приложения
из Google Play Store на Windows или macOS.
Бизнес-процесс:
Путь пользователя в данном приложении состоит из просмотра каталога
готовой продукции в буфете ИКИТ, его добавления в корзину и дальнейшей3
покупке. Также студент или преподаватель сможешь подробно посмотреть
состав и описание конкретного блюда или напитка.
